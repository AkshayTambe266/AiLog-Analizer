TROUBLESHOOTING
───────────────

If Ollama won't start:
  Get-Process ollama | Stop-Process -Force
  Start-Sleep 2
  ollama serve

If backend won't connect to Ollama:
  Check .env file has: OLLAMA_API_URL=http://localhost:11434
  Verify Ollama is running in Terminal 1
  Restart backend

If npm install fails:
  cd 'D:\OneDrive\Desktop\GitLab\AiLog Analizer\frontend'
  npm install
  npm start
